# Pontificia Universidad Javeriana- Procesamiento de datos #
## Taller-Python--Simón-Fajardo ##

Este repositorio contiene el desarrollo completo de nueve ejercicios prácticos orientados al aprendizaje progresivo de los fundamentos del lenguaje de programación Python.

El taller aborda estructuras de datos esenciales y estructuras de control, consolidando competencias básicas para el procesamiento y manipulación de información.

Los temas trabajados incluyen:

Cadenas (Strings)

Tuplas

Listas

Conjuntos

Diccionarios

Condiciones y estructuras de decisión

Cada notebook desarrolla ejercicios prácticos que permiten comprender tanto la sintaxis como la lógica computacional subyacente.


##Objetivo general ##
Desarrollar competencias fundamentales en programación con Python mediante la manipulación estructurada de datos y la aplicación de lógica condicional para la resolución de problemas.
